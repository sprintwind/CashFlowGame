# 现金流游戏 (CashFlow Game)

基于《富爸爸穷爸爸》的现金流游戏Android实现。

## 游戏目标

跳出"老鼠赛跑"，让被动收入超过支出！

## 功能特点

- 20格游戏棋盘
- 掷骰子移动
- 多种游戏卡片：
  - 职业卡 - 不同职业有不同的收入和支出
  - 投资机会 - 购买资产增加被动收入
  - 市场事件 - 收益或损失
  - 慈善捐款 - 增加被动收入
  - 生孩子 - 增加支出
  - 失业/创业 - 收入变化

## 游戏规则

1. 玩家从起点开始
2. 掷骰子决定移动步数
3. 到达不同格子触发相应事件
4. 每月获得现金流（收入 - 支出）
5. 购买投资机会增加被动收入
6. 当被动收入 > 支出时获胜

## 技术栈

- Kotlin
- Android SDK
- Material Design

## 编译

```bash
./gradlew assembleDebug
```

APK生成位置: `app/build/outputs/apk/debug/app-debug.apk`
